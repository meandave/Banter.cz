Why Learn Java? Learning Java 001

by David

Posted on March 8th, 2016 at 2:04 AM

How else to kickstart a series without explaining why I did so? I have been keeping away from Java for a long time, longer than neccessary.

Why? Complaints coming from everywhere about the verbosity of this language, and constant predictions of it's death. Well, I am pretty sure it is not going anywhere soon. Let's highlight a few facts why.

  • Cross platform portability thanks to JVM
  • Loads of established technologies using Java
  • Native language for developing apps for Android

I am learning Java because it will open many doors since it is so widely used.
